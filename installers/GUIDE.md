# Installers Documentation

## Comprehensive Setup Guides for All Platforms

This document provides step-by-step setup guides for installing FinApp on various platforms. Please follow the instructions relevant to your operating system.

---

### 1. Windows
#### Prerequisites
- Ensure you have Windows 10 or later.
- Install the latest versions of Git and Node.js.

#### Installation Steps
1. **Download Installer**: Go to the [Releases](https://github.com/nurchikoo19/finapp/releases) section and download the Windows installer.
2. **Run Installer**: Double-click the downloaded `.exe` file to run the installer.
3. **Follow Instructions**: Follow the on-screen instructions to complete the installation.
4. **Verify Installation**: Open Command Prompt and run `finapp --version` to verify the installation.

---

### 2. macOS
#### Prerequisites
- Ensure you have macOS Mojave or later.
- Install Homebrew if not already installed.

#### Installation Steps
1. **Download Installer**: Visit the [Releases](https://github.com/nurchikoo19/finapp/releases) section to download the macOS installer.
2. **Run Installer**: Open the `.dmg` file and drag the FinApp icon to your Applications folder.
3. **Launch Application**: Go to Applications and launch FinApp.
4. **Verify Installation**: Open Terminal and type `finapp --version` to verify the installation.

---

### 3. Linux
#### Prerequisites
- Ensure you have a package manager (APT, YUM, etc.) installed, depending on your distribution.

#### Installation Steps
1. **Download Installer**: Navigate to the [Releases](https://github.com/nurchikoo19/finapp/releases) and download the appropriate `.tar.gz` for your distribution.
2. **Extract Package**: Use the command `tar -zxvf finapp-x.y.z.tar.gz` to extract the package.
3. **Run Installer**: Inside the extracted folder, run the installation script with `sudo ./install.sh`.
4. **Verify Installation**: After installation, run `finapp --version` in the terminal to confirm installation.

---
