# Installation Instructions for FinApp

## Android Installation Instructions
1. **Download the APK**: Visit the releases section on GitHub to download the APK file for Android.
2. **Enable Installations from Unknown Sources**: Go to your device settings, find `Security`, and enable the option to allow installations from unknown sources.
3. **Install the APK**: Locate the downloaded APK file using a file explorer and tap on it to start the installation.
4. **Open FinApp**: Once installed, find the FinApp icon in your app drawer and open it.

## Windows Installation Instructions
1. **Download the Installer**: Visit the releases section on GitHub to download the Windows installer (.exe).
2. **Run the Installer**: Double-click the downloaded .exe file to start the installation process.
3. **Follow On-Screen Instructions**: Follow the prompts in the installer to complete the setup.
4. **Launch FinApp**: After the installation is complete, find FinApp in your Start Menu and launch it.
